# EVOX India WordPress Theme

An installable classic WordPress theme matching the EVOX India website design.

## Installation

1. Upload `evox-india.zip` in **Appearance → Themes → Add New → Upload Theme**.
2. Activate **EVOX India**.
3. Go to **Settings → Permalinks** and click **Save Changes** once.
4. Create pages for About, Industries, Contact, FAQ, Get Quote, Privacy Policy and Terms.
5. Set a static homepage under **Settings → Reading**.
6. Assign a menu to **Primary navigation** under **Appearance → Menus**.
7. Add products under **EVOX Products** and use a transparent product image as the Featured Image.

## Product fields

- Product type
- Hero eyebrow
- Hero statement
- Colour tone: `obsidian`, `porcelain`, `signal`, `slate`, or `sand`
- Quick specs: `Value | Label`
- Features: one per line
- Specifications: `Label | Value`

Contact details and the quote URL can be changed under **Appearance → Customize → EVOX contact details**.

The theme supports native WordPress title tags and works alongside common SEO plugins such as Yoast SEO, Rank Math or The SEO Framework.
